Farmer Assist System
About the Project

This is a complete web application developed using React. The main aim of the project is to help farmers access information about government schemes and apply for them easily.

The project is fully working and includes features like authentication, scheme listing, voice input, and language translation. It is designed to be simple and easy to use.

Features :
User registration and login
View available schemes
Apply for schemes
Voice input functionality
Multi-language support
Responsive UI
Technologies Used
React.js
JavaScript
Git & GitHub
npm

Libraries used:
react-i18next (for language translation)
Fetch API (for API communication)

How to Run the Project
Clone the repository
git clone https://github.com/mahi-laddha/Farmer-assistant.git
cd Farmer-assistant
Install dependencies
npm install --legacy-peer-deps
Run the project
npm start
Open in browser
http://localhost:3000
